# Markup Tip! 

### label 태그와 for 속성

* label 태그의 for 속성 
* input 태그의 id 속성

```
<input type="checkbox" id="subscribe">
<label for="subscribe">누르기</label>
```
label과 for 속성을 적절히 활용하면

input대신 label을 눌러도 input을 선택할 수 있다.

(input의 id, label의 for 속성을 똑같이 맞춰주면 됨)
